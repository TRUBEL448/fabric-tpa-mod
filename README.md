# TPA Mod - Fabric 1.21.4

Automatyczne wysyłanie komend `/tpa` co 1 minutę z ładnym GUI!

## Instalacja

1. **Klon repo i kompilacja:**
```bash
git clone https://github.com/TRUBEL448/fabric-tpa-mod.git
cd fabric-tpa-mod
./gradlew build
```

2. **Plik `.jar` będzie w:** `build/libs/tpa-mod-1.0.0.jar`

3. **Wrzuć `.jar` do:** `mods/` (Fabric folder)

## Użycie

1. **Otwórz GUI:** Naciśnij klawisz `T`
2. **Dodaj graczy:** Wpisz nick i kliknij "Dodaj"
3. **Usuń gracza:** Kliknij na niego i wciśnij "Usuń"
4. **Włącz TPA:** Kliknij "Start TPA"
5. **Wyłącz TPA:** Kliknij "Stop TPA"

Mod będzie automatycznie wysyłać `/tpa [nick]` co 1 minutę!

## Wymagania

- Minecraft 1.21.4
- Fabric Loader
- Java 21+
